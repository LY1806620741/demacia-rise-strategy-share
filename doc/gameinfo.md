# 资源

银盾
银盾用于建造和升级聚落，也能为王国
解锁研究升级。
在英雄联盟和云顶之弈完成特定任务可
获得银盾。

食物
训练王国内每个单位都需要1份食物。
建造农场，可扩大军队规模。

木材(+2664)
木材广泛用于建筑和单位的建造，以及
研究。
木材可通过伐木场生产。
聚落产出:+2327
护林场 +337

石料(+1469)
石料用于单位和建筑的升级。
石料可通过采石场生产。

金属(+1799)
金属用于高级单位的建造，以及研究。
金属可通过锻炉生产。
聚落产出:+1094
铸造命运+705

禁魔石(+23)
禁魔石用于最高级单位和建筑的建造，
以及研究。
禁魔石可通过禁魔石工场生产。

勇气
勇气是一种资源，可用于解锁王国与
单位的研究。
战斗胜利、升级聚落和解决特定事件
均可获得勇气。
