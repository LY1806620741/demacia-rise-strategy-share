# 快速开始指南 - 新增策略编辑器功能

## 🎮 5 分钟快速上手

### 第一步：了解界面布局

打开应用后，你会看到三个主要部分：

```
┌─────────────────────────────────────────┐
│  📊 状态仪表盘（顶部）                  │
│  显示：网络状态、节点数、数据统计       │
├─────────────────────────────────────────┤
│  ⭐ 官方推荐方案                        │
│  4 种推荐阵容，带"采用方案"按钮        │
├─────────────────────────────────────────┤
│  标签导航: ⚔️ | 🗺️ | 🔍                │
│  ├─ 战斗策略编辑                       │
│  ├─ 地图策略页面                       │
│  └─ 相似度推荐搜索                     │
└─────────────────────────────────────────┘
```

### 第二步：发布你的第一个战斗策略

**步骤 A：配置敌人阵容**
1. 点击 "⚔️ 战斗策略编辑" 标签
2. 从"选择敌人类型"下拉菜单选择一个敌人
3. 点击"+ 添加到敌人配队"
4. 可选：编辑敌人数量（右侧数字框）
5. 可选：添加更多敌人单位

**步骤 B：配置防守阵容**
1. 左侧看到"可用防守单位"列表
2. 将单位**拖拽**到右侧"已选应对单位"区域
3. 最多可选 8 个防守单位

**步骤 C：填写策略信息**
1. 输入**策略标题**，如："克制龙犬的最优方案"
2. 在**战术描述**中写入详细思路
3. 从**科技应对**下拉菜单选择相应科技
4. 点击**"发布战斗策略到德玛西亚网络"**

✅ 完成！你的策略已发布到整个网络！

### 第三步：搜索相似策略

**目标**：你遇到了一个"诺克萨斯龙犬"阵容，想找最佳应对方案

1. 点击 "🔍 相似度推荐搜索" 标签
2. **方式 A**（推荐）：
   - 在"选择预设敌人阵容"下拉菜单中选择"诺克萨斯龙犬队"
   
3. **方式 B**（自定义）：
   - 在"输入敌人阵容"框中输入: `诺克萨斯龙犬,诺克萨斯龙犬,部落战士`

4. 点击"🔍 搜索相似策略"

5. 查看结果：
   ```
   ┌─────────────────────────────┐
   │ 方案 1 - 相似度: 95% [高度] │
   │ 推荐: 卫兵×3 + 士兵×2       │
   │ [采纳此方案]                │
   └─────────────────────────────┘
   
   ┌─────────────────────────────┐
   │ 方案 2 - 相似度: 82% [高度] │
   │ 推荐: 卫兵×4 + 弓兵×2       │
   │ [采纳此方案]                │
   └─────────────────────────────┘
   ```

6. 点击"采纳此方案"应用到你的防守

## 📊 理解仪表盘

### 各指标含义

| 指标 | 含义 | 示例 |
|------|------|------|
| **网络状态** | 连接情况 | 🟢 已连接 / 🟡 单机 / 🔴 离线 |
| **P2P 节点数** | 对等网络节点 | 5 (你 + 其他 4 个用户) |
| **社区策略数** | 网络中同步的策略 | 127 (已收集的社区方案) |
| **本地保存数** | 你本机保存的策略 | 12 (你创建的方案) |
| **官方英雄数** | 游戏官方英雄 | 47 (当前版本英雄) |
| **官方建筑数** | 游戏官方建筑 | 32 (所有可建建筑) |

### 实时更新
指标每秒自动更新，反映最新状态。

## ❓ 常见问题

**Q: 为什么搜索没有结果？**
> 确保已创建或加载了至少一个策略。也可以先尝试采用官方推荐方案。

**Q: 敌人配队最多可以有几个？**
> 没有限制，但建议不超过 10 个以保持游戏平衡性。

**Q: 防守阵容最多可以选多少个单位？**
> 最多 8 个（系统限制），这也是游戏中的标准部队大小。

**Q: 我的策略会保存到本地吗？**
> 是的，会自动保存到浏览器的 IndexedDB。即使关闭浏览器，数据也不会丢失。

**Q: 离线时能使用吗？**
> 可以！离线时依然支持：
> - 创建本地策略
> - 本地全文搜索
> - 查看官方推荐
> - 当网络恢复时，本地策略会自动同步到 P2P 网络

**Q: 怎样清除我的所有本地数据？**
> 在浏览器开发者工具中找到 Application → IndexedDB → DemaciaRise，删除所有表。或者使用浏览器的"清除浏览数据"功能。

**Q: 推荐中的相似度是如何计算的？**
> 使用 Jaccard 相似度公式：
> ```
> 相似度 = |共同单位| / |所有不同单位|
> 示例：[A,B,C] vs [B,C,D] = 2/4 = 50%
> ```

**Q: 官方推荐和我的策略有什么区别？**
> - **官方推荐**：由游戏设计者提供，根据平衡性精心设计
> - **社区策略**：由玩家创建，可能更创新但也可能不平衡

**Q: 如何分享我的策略给其他玩家？**
> 发布后会自动通过 P2P 网络同步。其他在线用户会看到你的策略，并可以在搜索时找到它。

## 🚀 进阶使用

### 创建平衡的策略
1. 分析敌人阵容的构成
2. 选择克制单位的防守
