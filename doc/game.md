# 德玛西亚的崛起 小游戏

《德玛西亚的崛起》是英雄联盟客户端内的回合制策略小游戏，核心围绕敌人、兵种、英雄、科技树四大维度决定战斗胜负。下面按模块完整拆解：
一、主要敌人（威胁类型与克制）
敌人分野怪、部落势力、装甲怪物、暗裔军团四类，各有弱点与应对逻辑：
1. 野怪类（前期）
石甲虫：高近战减免 → 怕远程（弓兵 / 弩兵）
部落战士 / 狂战士：冲后排 → 用步兵 / 盾兵阻挡
巨魔：高单体伤害 → 用卫兵 / 盾兵拉仇恨
雪人：AOE 范围伤害 → 需前后排拉开站位
巨龙：对轻甲单位极高伤害 → 必须主力混合编队
2. 区域特殊敌人（中后期）
牧草谷：高防狼 + 枪兵 → 盾兵 + 长矛 + 弓兵
福斯拜罗 / 鹰石镇：装甲怪物 → 弩兵（破甲）+ 盾兵 + 莫甘娜
黎明要塞：密集长矛兵海 → 弩兵 + 琴女 + 加里奥
暗裔军团（剧情 BOSS）：剑魔亚托克斯带队 → 需凯尔 + 莫甘娜 + 禁魔石科技 counter
二、兵种体系（克制链与定位）
核心克制链：步兵 → 弓兵 → 骑兵 → 步兵；魔法兵 / 弩兵为特殊单位：
1. 基础兵种（科技解锁）
步兵 / 盾兵：前排坦度，抗伤、挡冲锋 → 克制骑兵、部落战士
骑兵：高机动，切后排 → 克制弓兵、远程单位
弓兵：基础远程，清野、打石甲虫 → 克制步兵
弩兵：破甲、百分比伤害 → 克制装甲怪物、高防单位（中后期核心）
2. 特殊 / 高级兵种
长矛兵：反骑、阵地战 → 克制骑兵、集群敌人
魔法兵：禁魔 / 魔法伤害 → 克制法师、暗裔单位（依赖禁魔石科技）
卫兵：稳定拉仇恨 → 克制巨魔、单体高伤怪
三、可用英雄（强度梯队与作用）
英雄为强力单位，提供控制、AOE、团队 buff、单体爆发，直接决定上限：
T0（必练 / 核心）
凯尔（天使）：攻速快、持续输出、AOE → 清场、打 BOSS 核心
莫甘娜：控制 + 魔法伤害 + 禁魔 → 克制暗裔、法师、集群敌人
拉克丝：大范围 AOE → 清理杂兵、破集群
T1（强力 / 常用）
琴女（娑娜）：团队攻击 / 血量加成 → 后排输出队核心辅助
加里奥：群体嘲讽、坦度 → 聚怪、保护后排
盖伦：前排坦度、单体斩杀 → 前期过渡，后期不如盾兵稳定
T2（功能 / 慎用）
波比：控制 + 保护 → 功能单一、容错低
嘉文 / 赵信：突击 / 前排 → 泛用性一般
四、科技树（影响战斗的关键升级）
科技分系统、王国、战斗三类，消耗木材、石头、勇气值（Valor），直接提升战力与资源效率：
1. 战斗科技（优先点）
基础军事：解锁步兵、弓兵、基础攻防
禁魔石技术：核心！克制魔法敌人、提升魔法兵伤害、削弱暗裔
高级兵种：解锁弩兵、长矛兵、骑兵
战场扩增：提升上场单位上限（最高 7 人）→ 阵容质变
兵种专精：步兵 / 弓兵 / 骑兵攻防、血量、克制加成
破甲技术：弩兵破甲率、百分比伤害提升（打装甲怪必备）
2. 王国科技（资源与扩张）
资源效率：伐木、采石、产银盾速度 → 快速爆兵
聚落扩张：解锁新城镇、增加人口上限
防御工事：城墙、瞭望塔 → 守城、减少战损
3. 系统科技（功能与奖励）
解锁英雄招募、事件奖励、赛季通行证进度
五、战斗胜负核心逻辑（总结）
敌人克制优先：石甲虫用远程、装甲怪用弩兵、暗裔用禁魔 + 凯尔 / 莫甘娜
兵种搭配：前排盾兵 / 步兵 + 后排弓兵 / 弩兵 + 骑兵切后 + 魔法兵 counter 魔法
英雄组合：T0 核心（凯尔 + 莫甘娜 + 拉克丝）+ 辅助（琴女）+ 前排（加里奥）
科技碾压：优先点禁魔石、战场扩增、破甲、高级兵种，再补资源与防御
