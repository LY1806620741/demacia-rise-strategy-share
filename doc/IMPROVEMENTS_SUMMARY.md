### 2. 🆕 官方推荐面板
**功能**: 基于豆包报告给出的游戏平衡建议

#### 推荐方案（基于豆包报告）
1. **对抗诺克萨斯龙犬**
   - 难度等级: ⭐⭐⭐⭐
   - 推荐防守: 卫兵 × 3 + 士兵 × 2
   - 原因: 龙犬是远程闪避单位，近战克制效果最佳

2. **对抗部落战士**
   - 难度等级: ⭐⭐⭐
   - 推荐防守: 士兵 × 5 + 卫兵 × 3
   - 原因: 部落战士克制远程，需要坦克防线

3. **对抗龙蜥飞行单位**
   - 难度等级: ⭐⭐⭐⭐⭐
   - 推荐防守: 游侠 × 3 + 盖伦 × 1
   - 原因: 飞行单位需要特定克制，游侠单体爆发最强

4. **均衡防守方案**
   - 难度等级: ⭐⭐⭐
   - 推荐防守: 卫兵 × 4 + 弓兵 × 2 + 加里奥 × 1
   - 原因: 通用高效方案，适合大多数场景

### 5. 🆕 相似度推荐搜索
**功能**: 根据敌人阵容推荐最佳防守策略

#### 搜索方式
**方式 A: 预设敌人阵容选择**
```
选择预设敌人阵容 (下拉菜单)
  ├─ 诺克萨斯龙犬 (威胁度: ⭐⭐⭐⭐)
  ├─ 部落战士 (威胁度: ⭐⭐⭐)
  └─ 龙蜥飞行单位 (威胁度: ⭐⭐⭐⭐⭐)
```

**方式 B: 手动输入敌人阵容**
```
输入格式: 单位名称,单位名称,单位名称 (逗号分隔)
示例: 诺克萨斯龙犬,诺克萨斯龙犬,部落战士
```

#### 推荐算法
```rust
// 在 search.rs 中实现的相似度计算公式
similarity_score = Jaccard(敌人阵容, 策略目标) × 0.6 
                  + 策略评分_boost × 0.4

// Jaccard 相似度: |交集| / |并集|
// 策略评分_boost: (评分 / 50).min(1.0)
```

**相似度解释：**
- **70% 以上**: 🟢 高度匹配 - 强烈推荐
- **50-70%**: 🟡 部分匹配 - 可参考
- **以下 50%**: 参考用，可能需要调整

#### 推荐结果展示
```
┌─────────────────────────────────┐
│ 方案 1 - 相似度: 85% | [高度匹配]│
│ 推荐阵容: 卫兵×3 + 士兵×2       │
│ [采纳此方案]                    │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ 方案 2 - 相似度: 72% | [高度匹配]│
│ 推荐阵容: 卫兵×4 + 弓兵×2       │
│ [采纳此方案]                    │
└─────────────────────────────────┘
```

## 📊 数据流示意图

```
用户界面
  ├─ 敌人配队 (enemy_queue)
  │   └─ [{id, name, quantity}, ...]
  │
  ├─ 应对阵容 (counter_units)
  │   └─ [Unit, Unit, Unit, ...]
  │
  └─ 搜索查询 (similarity_search)
      ├─ WASM: calculate_lineup_similarity()
      ├─ WASM: recommend_counters()
      └─ 返回 {strategy_id, counter_lineup, similarity_score}

存储层
  ├─ IndexedDB: strategies (本地)
  ├─ IndexedDB: lineups (预设方案)
  └─ P2P 网络: STRATEGIES[] (社区同步)
```

## 🎮 使用流程

### 流程 A: 发布新的战斗策略
```
1. 点击"⚔️ 战斗策略编辑"标签
2. 第一步 - 配置敌人阵容：
   - 选择敌人单位
   - 编辑数量
3. 第二步 - 配置应对阵容：
   - 拖拽防守单位到"已选应对单位"区域
4. 填写策略信息：
   - 标题 (如: "克制龙犬+马扎阵容")
   - 详细描述
   - 科技应对
5. 点击"发布战斗策略到德玛西亚网络"
```

### 流程 B: 查询敌人相似策略
```
1. 点击"🔍 相似度推荐搜索"标签
2. 方式选择：
   A) 从"选择预设敌人阵容"列表选择
   B) 或手动输入敌人阵容 (逗号分隔)
3. 点击"🔍 搜索相似策略"
4. 查看推荐结果（按相似度降序排列）
5. 点击"采纳此方案"应用推荐
```

### 流程 C: 查看官方推荐方案
```
1. 打开应用首页
2. 在"⭐ 官方推荐方案"面板查看
3. 每个方案包含：
   - 对敌类型
   - 推荐防守阵容
   - 难度评级
   - "采用方案"按钮
4. 点击按钮应用到战斗编辑器
```

## 🌐 P2P 网络集成

- **节点发现**: localStorage 中存储已知节点
- **策略同步**: 使用 JSON 序列化的 WildStrategy
- **心跳机制**: 每 3 秒发送一次心跳维持连接
- **节点 TTL**: 10 秒超期清理未响应节点
