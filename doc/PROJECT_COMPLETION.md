#### 4️⃣ 相似度推荐引擎
```
Jaccard 相似度算法
├─ 单位集合交集 ÷ 并集
├─ 支持 60% 敌人相似度权重
├─ 支持 40% 策略评分权重
└─ 最终分数: 0-1.0

示例计算:
敌人: [龙犬, 龙犬, 战士]
策略 A: [龙犬, 龙犬, 坦克]
相似度 = 2/4 × 0.6 + 评分加成 × 0.4 = 42-70%
```
